
# Fin Dashboard (Vite + React + Tailwind)

This is a minimal Vite + React + Tailwind project containing the Fin Dashboard app you provided.

## Quick start (locally)
1. Install dependencies:
   ```
   npm install
   ```
2. Run dev server:
   ```
   npm run dev
   ```
3. Open the URL printed by Vite (usually http://localhost:5173).

## Build
```
npm run build
npm run preview
```

## Deploy
You can deploy to Vercel, Netlify, or GitHub Pages. For Vercel:
1. Install Vercel CLI (`npm i -g vercel`) or connect repository in Vercel dashboard.
2. Run `vercel` in project root and follow prompts.
